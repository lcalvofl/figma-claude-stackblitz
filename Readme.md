# Figma → Claude → StackBlitz

Repository accompanying my LinkedIn experiment.

## Goal

Test whether React code generated by Claude from a Figma component can run with minimal manual intervention.

## Steps

1. Create a new React project in StackBlitz.
2. Replace `App.tsx` with the file provided.
3. Add `DatePicker.jsx` to the `src` folder.
4. Run.

That's it.
